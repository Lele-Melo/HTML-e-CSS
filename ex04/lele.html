Vamos colocar imagens ✨

Para a primeira forma vc deve baixar a imagem e colocar dentro da pasta ex04
Agora para adicionar basta colocar:
1 - img+enter
2 - Dentro das aspas ctrl+espaço
3 - selecionar a imagem e colocar uma legenda

Para a segunda forma de colocar vc deve abrir a imagem na guia do google e copiar o endereço da imagem
Agora para adicionar basta colocar:
1 - img+enter
2 - Dentro das aspas colar a imagem e colocar uma legenda
Obs: a imagem so vai aparecer se o site de onde você pegou estiver funcionando
